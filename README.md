# Intro
This is a Clicking Game that test your endurance and finger strength(yes by click a bunch of boxes)

# How To Start
1.Unity Play URL(https://play.unity.com/en/games/03ed8253-59f2-48ae-be8f-484a4fe9edd4/box-clicker)
2. from build and access Box-Clicker.exe

# How to Play
1.Click The Box
2.Click Power Up(slow box fall or double score)
3.Dont Let the box fall or you will lose Live
